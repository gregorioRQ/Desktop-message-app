# MSG Desktop 

Aplicación de mensajería.

## Funcionalidades:
 - Mensajes en tiempo real.
 - Administración de contactos.
 - Notificaciones.
 - Login y registro.

## Tecnologías y arquitecturas

### Cliente
- JavaFX (UI)
- SQLite (Base de datos local del usuario)
- Maven (Gestión del proyecto)


### Backend
- Java (jdk 21)
- Springboot (Framework)
- MySQL (Base de datos de los distintos servicios)
- Spring Security y JWT (Validación y autorización)
- Spring Data JPA (Orm)
- RabbitMQ (Colas notificaciones)
- Docker 
- Junit y Mockito (Pruebas)
- WebSockets y STOMP (Comunicación en tiempo real)
- Protobuf 
- Microservios
- Event Driven
- Maven
